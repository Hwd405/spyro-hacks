Spyro: Year of the Dragon Randomiser by Hwd405
----------------------------------------------

Contained in this release:
* SpyroYotDRandomiser.exe - the randomiser
* dumpsxiso.exe - Lameguy64's iso extraction tool
* mkpsxiso.exe - Lameguy64's iso creation tool
* README.txt - this file

----------------------------------------------

The easiest method to run the randomiser is simply to drag and drop the .bin / .iso file of your choosing onto SpyroYotDRandomiser.exe. Then, simply follow the instructions shown in the command window.

If you wish to enter a seed, this will need to be done via the command prompt. Open a blank command prompt window in your working directory of choice (e.g. by entering "cmd" into the address bar in the folder that contains the randomiser tool) and enter a command taking the following form:
SpyroYotDRandomiser.exe <filePath> <seed>
* filePath - the path to the copy of Spyro 3 you wish to use as a base
  * I recommend placing the Spyro 3 copy in the same file as the randomiser, so that a local file reference can be used (e.g. `./spyro3.bin`) rather than requiring you enter a full file path
* seed - a number to pass in as randomiser seed
  * Some seeds will generate a "hard mode" version of the randomiser - any seed ending "405" will activate this mode!

An example command to run is given below, assuming that the copy of the game has the filename "spyro3.bin" and the seed "1000" is being used:
SpyroYotDRandomiser.exe ".\spyro3.bin" 1000

PAL versions of the game are not currently supported. Additionally, attempting to randomise an already randomised file will fail.

This is an experimental tool that still has some bugs. If you encounter any crashes, have any suggestions, or feel that certain aspects of the gameplay aren't perfect, please let me know.

Please also note that some egg locations generated by the randomiser are experimental and may cause strange behaviours and interactions with other mobys, or may render incorrectly. If an egg has been found but cannot be obtained, this is usually an indication that an enemy near to its original spot needs to be killed first.
